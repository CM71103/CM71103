## 🚀 Key Projects

### 🔍 DSA-Based Search Engine
*   **What it is:** A fast, Java-based search platform engineered using advanced data structures.
*   **Under the hood:** Implemented **Bloom Filters** for instant membership checking and **Prefix Arrays (Tries)** for ultra-fast query autocomplete.

### 🍱 Canteen Management System
*   **What it is:** A robust, high-throughput database system designed to handle high-volume ordering workflows.
*   **Under the hood:** Powered by **PostgreSQL**, fully normalized up to **BCNF** to completely eliminate data redundancy and minimize heavy query execution times.
